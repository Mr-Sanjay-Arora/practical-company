--- 1. Setup Project ---
Extract all the files from the zip & save it into your folder
Create a database named company in phpmyadmin
Run the project using the command: php artisan serve

--- 2. Database File with sample data (.sql) ---
The database file which includes the sample data for this project has been added to the folder

--- 3. Links/Routes ---
All the routes have been defined the web.php file which is located in the routes folder
Links being generated in the webpages are passing through routes which in turn call the controller methods being invoked
